Projeto 1

Este projeto possibilita a visualização de uma planilha com mais de 400 mil linhas por meio de gráficos simples e intuitivos, facilitando a análise e interpretação dos dados. Para o desenvolvimento, foram utilizadas as seguintes tecnologias: Python para a lógica de programação, Pandas para manipulação e tratamento dos dados, Matplotlib para a criação dos gráficos e Flask para a construção da interface web.
